# HangHive Backend

Backend service for the HangHive application, built with Node.js, Express, and Prisma.

## Project Structure

```
HangHive_backend/
├── backend/
│   ├── src/
│   │   ├── controllers/
│   │   ├── middleware/
│   │   ├── routes/
│   │   ├── services/
│   │   ├── socket/
│   │   ├── utils/
│   │   └── index.js
│   ├── prisma/
│   ├── .env
│   └── package.json
├── functionality.md
└── README.md
```

## Getting Started

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the server:
   ```bash
   npm run dev
   ```
